# Proyecto de Ninjas

Sistema CRUD de ninjas inspirado en el universo de Naruto, desarrollado en Python puro. El proyecto permite registrar, consultar, actualizar y eliminar ninjas, utilizando árboles binarios para optimizar la organización y búsqueda de datos durante las batallas, además de actualizar sus atributos dependiendo de si gana o no la batalla.
